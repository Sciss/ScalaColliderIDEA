h3. requirements / installation

You can use @sbt update@ to install the ScalaCollider dependancy. currently compiles against Scala 2.8.1, and uses IDEA CE 10.0.2, and latest scala plugin.

The following files are expected in lib. Here are my symlinks for reference:
